About

This is a personal project to practice my design, CSS & javascript skills. I'm not necessarally a poetry fan but it seemed like a interesting subject to work with.

All poems are fetched from the PoetryDB API and displayed using vanilla javascript. The rest is build with good old HTML & CSS. Explore the code on Github.

The text is set in Garamond Premier Pro regular & italic. A classic serif typeface with lots of opentype features to play with.

All icons used in the interface are feather icons. Except for the back icon on this page wich is a modified back arrow icon.
